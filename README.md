
# Installation

1. Clone the repository:

git clone https://github.com/yulialavruk/dashboard-grid.git

cd dashboard-grid

2. Install dependencies:

npm install

3. Run the development:

npm run dev

# Deployment

The project is deployed on **GitHub Pages**.

**Live Demo:** [https://yulialavruk.github.io/dashboard-grid/](https://yulialavruk.github.io/dashboard-grid/)

Run the deploy:

npm run deploy
